# 🚗 AutoSchool Pro — Система управления автошколой


> Курсовой проект 2 курса | Desktop-приложение + Telegram-бот

---
## 📋 О проекте

**Instructor+** — это комплексное решение для автоматизации работы автошколы, состоящее из двух компонентов, работающих с общей базой данных.

(AutoAdmin) - резервный вариант.

### 🖥️ Desktop-приложение

Рабочее место **администратора** и **инструктора** автошколы. Реализуется на **C#** с использованием фреймворка **Avalonia UI** (кроссплатформенный desktop).
### 📱 Telegram-бот

Клиентская часть для **учеников**. Реализуется на **Python** с использованием фреймворка **aiogram**. Позволяет получать расписание, уведомления и записываться на занятия.

---
## 🎯 Цель проекта

Заменить текущий неструктурированный процесс (публикация времени и места занятий в общем Telegram-чате) на централизованную систему с:

- Единой базой данных всех сущностей автошколы
- Автоматической рассылкой расписания через бота
- Таргетированной записью (например, ученики с малым количеством уроков)
- Полноценным учётом документов, филиалов, автомобилей и инструкторов

## 🗄️ База данных (MySQL / MariaDB)

### Сущности для хранения:

- **Филиалы** автошколы
- **Администраторы** (учётные данные для входа в приложение)
- **Инструкторы** (учётные данные, привязка к филиалу и автомобилю)
- **Ученики** (личные данные, Telegram ID для связи с ботом)
- **Автомобили** (привязка к филиалу и инструктору)
- **Документы** учеников (тип документа, путь к файлу, срок действия)
- **Занятия / уроки** (дата, время, ученик, инструктор, автомобиль, филиал, статус)
- **Отмены занятий** (причина, кто отменил, дата отмены)
